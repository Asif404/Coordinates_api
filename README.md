# Coordinates_api

This Project uses FastApi Python framework. 

## Getting Started 

Clone the Repository

` git clone https://github.com/Asif404/Coordinates_api.git`

`python3 requirements.txt`

Add api key in main.py file

Run code `$ uvicorn main:app --reload`
